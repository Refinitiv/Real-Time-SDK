1.4
Definitions
 
Table 1: Definitions  
Definition
Description
Group
A related set of configuration parameters for a specific EMA component (e.g., ChannelGroup).
List
A list of components belonging to a group (e.g., ChannelList).
Component
A specific component (e.g., Channel). Because lists can have multiple components, each component must have a ‘name’ field for identification purposes.
Field
A configurable parameter.
Default Value
A default value is the value the API uses if a value is not specified by the user. In general, items with default values are required by the API.
Allowed value
Specific values or a range of values that the field allows.
