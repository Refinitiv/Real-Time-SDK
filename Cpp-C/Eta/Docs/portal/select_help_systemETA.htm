Transport API C Edition - Integrated Web Documentation

This web documentation is a new integrated single entry point web-based help system containing all of the Transport API C Edition product’s informational content.

This help system supports only select browsers. See product readme for qualified/supported browsers. If you run into a problem with your browser, please use the “classic style” documentation.

This deployment continues to ship the “classic style” documentation for users of browsers that don’t support the new web based documentation.
