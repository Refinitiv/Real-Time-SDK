Description
Page size (in rows and columns) is maintained statically by the Triarch ANSI Library. (It can be set using the qa_set_rows() and qa_set_columns() function calls.) qa_page_rows() returns the current number of rows in a page.
The Triarch ANSI Library uses the statically maintained number of rows in a page to process page images during the qa_encode(), qa_decode(), and qa_reset() function calls. The information is used to determine the size of the CHARTYP buffer being passed to those functions.
