Response Message Payload
The Response Message (refer to Section 2.3) payload can vary widely, based on its DictionaryType. The payload is typically a of s, but can also be XML or Opaque data. For further details on the response message payloads, refer to Section 5.5.1 and Section 5.6.1.
Some DictionaryTypes also have external file representations of their data. For details about the data of each DictionaryType, refer to Section 5.5.2 and Section 5.6.2.
