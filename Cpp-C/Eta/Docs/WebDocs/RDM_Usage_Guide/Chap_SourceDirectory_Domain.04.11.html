Source Directory Group Filter Entry
The Group filter entry (RDM_DIRECTORY_SERVICE_GROUP_FILTER, RDM_DIRECTORY_SERVICE_GROUP_ID) conveys item group status and item group merge information. Every item stream is associated with an item group as defined by the groupId provided with the itemâ€™s RsslRefreshMsg or RsslStatusMsg. If some kind of change impacts all items within the same group, only a single group status message need be provided. For more information on item group use and handling, see the Transport API C Edition Developers Guide.
If multiple Group RsslFilterEntrys are received in a single RsslFilterList, then they should be applied in the order in which they were received.
