Login Generic Message Payload
The Login data structure for GenericMsg.Payload is a Map of AsciiString->ElementList. Each key is a ServiceName.Map.Key. Each RsslElementList contains one RsslElementEntry. The Login GenericMsg.Payload is formatted as an RsslMap type, with a keyPrimitiveType of RSSL_DT_ASCII_STRING and a containerType of RSSL_DT_ELEMENT_LIST. There is no summary data and typically only one map entry that informs the provider of its warm standby role. Specific information is contained in the tables below.
