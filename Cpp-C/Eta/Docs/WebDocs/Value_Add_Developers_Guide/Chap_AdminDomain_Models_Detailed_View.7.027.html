RSSL RDM Login Consumer Connection Status Structure Members
 
Table 100: RsslRDMLoginConsumerConnectionStatus Structure Members  
STRUCTURE MEMBER
DESCRIPTION
flags
Required. Indicate the presence of optional login consumer connection status members. For details, refer to Section 7.3.5.2.
rdmMsgBase
Required. Contains general message information like streamId and domainType. Indicates the Login Message type (for login connection status, set to LoginMsgType.CONSUMER_CONNECTION_STATUS).
warmStandbyInfo
Optional. Includes RsslRDMLoginWarmStandbyInfo to convey the state of the upstream provider. For details, refer to Section 7.3.5.3.
If present, a flags value of RDM_LG_CCSF_HAS_WARM_STANDBY_INFO should be specified.
