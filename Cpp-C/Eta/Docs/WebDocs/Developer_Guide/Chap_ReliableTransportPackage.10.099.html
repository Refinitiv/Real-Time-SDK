rsslServerIoctl Option Values
The following table provides a description of the options available for use with the rsslServerIoctl function.
 
Table 68: rsslServerIoctl Option Values  
OPTION ENUMERATION
DESCRIPTION
RSSL_SERVER_NUM_POOL_BUFFERS
Allows an RsslServer to change its sharedPoolSize setting. Value should be pointer to RsslUInt32
RSSL_SERVER_PEAK_BUF_RESET
Allows an RsslServer to reset the peakBufferUsage statistic. Value is not required.
