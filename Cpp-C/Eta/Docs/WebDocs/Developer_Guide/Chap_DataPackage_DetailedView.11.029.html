Transport API C Edition

Utility Functions
The Transport API provides the following utility functions for use with RsslState:
 
Table 89: RsslState Utility Functions  
FUNCTION NAME
DESCRIPTION
rsslClearState
Sets all members in RsslState to an initial value. This includes setting streamState to its unspecified value, which is not intended to be encoded or decoded.
rsslIsFinalState
•
Returns true if the RsslState represents a final state for a stream (e.g. stream is Closed, Closed Recover, Redirected, or NonStreaming).
•
Returns false if the RsslState is not final.
rsslStreamStateToString
Converts an RsslState.streamState enum to a string representation.
rsslDataStateToString
Converts an RsslState.dataState enum to a string representation.
rsslToString
Converts an RsslState.code enum to a string representation.
rsslStreamStateInfo
Converts an RsslState.streamState enum to text (e.g., RSSL_DATA_OK to “Data state is OK”) and returns as a string for the streamState.
rsslDataStateInfo
Converts an RsslState.dataState enum to text (e.g., RSSL_DATA_OK to “Data state is OK”) and returns as a string for the dataState.
rsslStateCodeInfo
Takes an RsslState.code enum to text (e.g., RSSL_DATA_OK to “Data state is OK”) and returns as a string for the code.
