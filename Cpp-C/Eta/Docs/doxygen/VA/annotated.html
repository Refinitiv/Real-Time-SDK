Data Structures
Here are the UPA C Edition data structures with brief descriptions:
RsslCacheErrorError structure passed to some payload cache interface methods. Populated with information if an error occurs during the function call
RsslClassOfServiceUsed to configure and indicate negotiated parameters of a qualified stream
RsslClassOfServiceAuthenticationRepresents Authentication class of service properties used within a qualified stream
RsslClassOfServiceCommonRepresents Common class of service properties used within a qualified stream
RsslClassOfServiceDataIntegrityRepresents Data Integrity class of service properties used within a qualified stream
RsslClassOfServiceFlowControlRepresents Flow Control class of service properties used within a qualified stream
RsslClassOfServiceGuaranteeRepresents Guarantee class of service properties used within a qualified stream
RsslConsumerWatchlistOptions
RsslCreateReactorOptionsConfiguraion options for creating an RsslReactor
RsslErrorInfoError Information. Used by RsslRDMMsg encode and decode functions, as well as the RsslReactor
RsslHashLink
RsslHashTable
RsslMemoryBlock
RsslMemoryPool
RsslMsgEventBase event provided to Message Callback functions. This structure is provided to defaultMsgCallback functions and is present in structures provided to other Message Callback functions such as loginMsgCallback and directoryMsgCallback
RsslPayloadCacheConfigOptionsConfiguration options used when creating a payload cache instance
RsslRDMAddressPortInfoThe Address Port Info. Contains Host Address and Port information
RsslRDMConsumerStatusServiceInformation about how a Consumer is using a particular service with regard to Source Mirroring
RsslRDMDictionaryCloseThe RDM Dictionary Close. Used by an OMM Consumer to close a Dictionary stream
RsslRDMDictionaryMsgThe RDM Dictionary Msg. The Dictionary Message encoder and decoder functions expect this type. It is a group of the classes of message that the Dictionary domain supports. Any specific message class may be cast to an RsslRDMDictionaryMsg, and an RsslRDMDictionaryMsg may be cast to any specific message class. The RsslRDMMsgBase contains members common to each class that may be used to identify the class of message
RsslRDMDictionaryMsgEventEvent provided to Dicionary Message Callback functions
RsslRDMDictionaryRefreshThe RDM Dictionary Refresh. Used by an OMM Provider to provide the content of a Dictionary.
NOTE: The Field dictionary type supports being split into fragments, each containing some of the fields. rsslEncodeRDMDictionaryMsg() may need to be called multiple times to send the entire message. The startFid parameter will be updated with each call
RsslRDMDictionaryRequestThe RDM Dictionary Request. Used by an OMM Consumer to request a dictionary from a services that provides it
RsslRDMDictionaryStatusThe RDM Dictionary Status. Used by an OMM Provider to indicate changes to the Dictionary stream
RsslRDMDirectoryCloseThe RDM Directory Close. Used by an OMM Consumer to close an open Source Directory stream
RsslRDMDirectoryConsumerStatusThe RDM Directory Consumer Status. Used by an OMM Consumer to send Source Mirroring information
RsslRDMDirectoryMsgThe RDM Directory Msg. The Directory Message encoder and decoder functions expect this type. It is a group of the classes of message that the Directory domain supports. Any specific message class may be cast to an RsslRDMDirectoryMsg, and an RsslRDMDirectoryMsg may be cast to any specific message class. The RsslRDMMsgBase contains members common to each class that may be used to identify the class of message
RsslRDMDirectoryMsgEventEvent provided to Directory Message Callback functions
RsslRDMDirectoryRefreshThe RDM Directory Refresh. Used by an OMM Provider to provide information about available services
RsslRDMDirectoryRequestThe RDM Directory Request. Used by an OMM Consumer to request Source Directory information from an OMM Provider
RsslRDMDirectoryStatusThe RDM Directory Status. Used by an OMM Provider to indicate changes to the Directory stream
RsslRDMDirectoryUpdateThe RDM Directory Update. Used by an OMM Provider to provide updates about available services
RsslRDMLoginCloseThe RDM Login Close. Used by an OMM Consumer or OMM Noninteractive Provider to close a login
RsslRDMLoginConsumerConnectionStatusThe RDM Login Consumer Connection Status. Used by an OMM Consumer to send Warm Standby information
RsslRDMLoginMsgThe RDM Login Msg. The Login Message encoder and decoder functions expect this type. It is a group of the classes of message that the Login domain supports. Any specific message class may be cast to an RsslRDMLoginMsg, and an RsslRDMLoginMsg may be cast to any specific message class. The RsslRDMMsgBase contains members common to each class that may be used to identify the class of message
RsslRDMLoginMsgEventEvent provided to Login Message Callback functions
RsslRDMLoginRefreshThe RDM Login Refresh. Used by an OMM Provider to accept or reject a login request and provide information about features supported
RsslRDMLoginRequestThe RDM Login Request. Used by an OMM Consumer or OMM Noninteractive Provider to request a login
RsslRDMLoginStatusThe RDM Login Status. Used by an OMM Provider to indicate changes to the Login stream
RsslRDMLoginWarmStandbyInfoWarm Standby Information for the RDM Login Consumer Connection Status
RsslRDMMCAddressPortInfoThe MC Address Port Info. Contains Host Address Port and domain information
RsslRDMMsgRSSL Message Union contains all messages. This includes constructs, functions, and values
RsslRDMMsgBaseThis message structure contains the information that is common across all RDM Message formats. It is included in all Value Added RDM Components
RsslRDMQueueAckThe RDM Queue Ack
RsslRDMQueueCloseThe RDM Queue Close
RsslRDMQueueDataThe RDM Queue Data
RsslRDMQueueDataExpiredThe RDM Queue Data Expired Message. NOTE: Encoding this message is not supported
RsslRDMQueueMsgThe RDM Queue Msg. The Queue Message encoder and decoder functions expect this type. It is a group of the classes of message that the Queue domain supports. Any specific message class may be cast to an RsslRDMQueueMsg, and an RsslRDMQueueMsg may be cast to any specific message class. The RsslRDMMsgBase contains members common to each class that may be used to identify the class of message
RsslRDMQueueRefreshThe RDM Queue Refresh
RsslRDMQueueRequestThe RDM Queue Request. Used by an OMM Consumer to connect to a QueueProvider
RsslRDMQueueStatusThe RDM Queue Status
RsslRDMServerInfoInformation about an available Server. A list of RsslRDMServerInfo is used by an RsslRDMLoginRefresh to list servers available for connecting to, and whether to use them as Standby servers
RsslRDMServiceThe RDM Service. Contains information about a particular service
RsslRDMServiceDataThe RDM Service Data. Contains information provided by the Source Directory Data filter
RsslRDMServiceGroupStateThe RDM Service Group State. Contains information provided by the Source Directory Group filter
RsslRDMServiceInfoThe RDM Service Info. Contains information provided by the Source Directory Info filter
RsslRDMServiceLinkThe RDM Service Link. Contains information about an upstream source associated with the service
RsslRDMServiceLinkInfoThe RDM Service Link Info. Contains information provided by the Source Directory Link filter
RsslRDMServiceLoadThe RDM Service Load. Contains information provided by the Source Directory Load filter
RsslRDMServiceSeqMCastInfoThe RDM Sequenced Multicast Info. Contains information provided by the Source Directory Sequenced Multicast filter
RsslRDMServiceStateThe RDM Service State. Contains information provided by the Source Directory State filter
RsslReactorThe Reactor. Applications create RsslReactor objects by calling rsslCreateReactor, create connections by calling rsslReactorConnect/rsslReactorAccept and process events by calling rsslReactorDispatch
RsslReactorAcceptOptionsConfiguraion options for creating an RsslReactor server-side connection
RsslReactorAcceptTunnelStreamOptionsOptions for accepting a tunnel stream from a consumer
RsslReactorChannelChannel representing a connection handled by an RsslReactor
RsslReactorChannelEventAn event that has occurred on an RsslReactorChannel
RsslReactorChannelInfoStatistics returned by the rsslReactorGetChannelInfo() call
RsslReactorChannelRoleThe Reactor Channel Role structure. The rsslReactorConnect() and rsslReactorAccept() functions expect this structure. It is a group of the types of roles that are supported by RsslReactorChannels. The RsslReactorChannelRoleBase is used to identify the type of role
RsslReactorChannelRoleBaseBase of ReactorChannel role structures
RsslReactorConnectInfo
RsslReactorConnectOptionsConfiguraion options for creating an RsslReactor client-side connection
RsslReactorDispatchOptionsOptions for dispatching from an RsslReactor
RsslReactorOMMConsumerRoleStructure representing the role of an OMM Consumer
RsslReactorOMMNIProviderRoleStructure representing the role of an OMM Noninteractive Provider
RsslReactorOMMProviderRoleStructure representing the role of an OMM Provider
RsslReactorRejectTunnelStreamOptionsOptions for rejecting a tunnel stream from a consumer
RsslReactorRequestMsgOptionsOptions when using rsslReactorSubmitMsg. Provides simple methods of performing advanced item request behaviors such as batch requests and requesting views(the application may also request these behaviors by encoding them per the appropriate model, however these options may be more convenient). These options are supported only when a watchlist is enabled unless indicated otherwise. See rsslReactorSubmitMsg, RsslReactorSubmitMsgOptions
RsslReactorSubmitMsgOptions
RsslReactorSubmitOptionsOptions for submitting a message
RsslStreamInfoInformation about the stream associated with a message. Only used when a watchlist is enabled
RsslTunnelStreamRepresents a tunnel stream. A tunnel stream is a private stream that supports interactions with guaranteed messaging and other auxiliary services
RsslTunnelStreamAuthInfoAn event containing authentication information for an RsslTunnelStreamStatusEvent
RsslTunnelStreamCloseOptionsThe options for closing a TunnelStream
RsslTunnelStreamGetBufferOptionsOptions for getting a buffer for sending a message through the tunnel stream
RsslTunnelStreamMsgEventAn event indicating a message received in this tunnel stream
RsslTunnelStreamOpenOptionsThe options for opening a TunnelStream
RsslTunnelStreamQueueMsgEventAn event indicating a queue message received in this tunnel stream
RsslTunnelStreamRequestEventAn event indicating a new tunnel stream from a consumer
RsslTunnelStreamStatusEventAn event regarding the state of the tunnel stream
RsslTunnelStreamSubmitMsgOptionsOptions for submitting a message to a tunnel Stream
RsslTunnelStreamSubmitOptionsOptions for submitting a buffer to a tunnel Stream
