Transport API Data Primitive Types
Transport API Wire Format Data Package

The Transport API Primitive Types are common typedef and structure definitions used to represent simple content like integers or length specified buffers. These are available for use by other packages and applications alike and are required by the Transport API Data, Transport API Message, Transport API Dictionary and Transport API Transport packages.

 System Types
 

The system types are platform independent representations of system types (e.g. 8 bit signed integer, 16 bit unsigned integer, etc.).


 Transport API System Primitive Encoder Functions
 

These functions are used to encode selected system primitive types using an RsslEncodeIterator .


 Generic Primitive Type Utilities
 

Detailed information about generic functions that can help encode, decode, or convert many primitive types.


 RsslInt Reference Group
 

The integer primitive type allows the user to represent signed integer content ranging from -263 to 263 - 1.


 RsslUInt Reference Group
 

The unsigned integer primitive type allows the user to represent unsigned integer values ranging from 0 to 264 - 1.


 RsslReal Reference Group
 

The RsslReal Primitive Type is an optimized wire representation of a decimal or fractional value.


 RsslFloat Reference Group
 

The float primitive type allows the user to represent 4-byte IEEE-754 floating point values.


 RsslDouble Reference Group
 

The double primitive type allows the user to represent 8-byte IEEE-754 floating point values.


 RsslBuffer Reference Group
 

The buffer primitive type allows the user to represent length specified content.


 RsslDate Reference Group
 

The RsslDate Primitive Type is an optimized wire representation of date that can contain month, day, and year.


 RsslTime Reference Group
 

The RsslTime Primitive Type is an optimized wire representation of time that can contain hour, minute, second, and millisecond.


 RsslDateTime Reference Group
 

The RsslDateTime Primitive Type is an optimized wire representation of time that can contain month, day, year, hour, minute, second, and millisecond.


 RsslEnum Reference Group
 

The enum primitive type allows the user to represent an unsigned 2-byte value, typically corresponding to an enumerated value that can be cross-referenced with an enum dictionary.


 RsslArray Reference Group
 

The Array primitive type allows the user to represent a uniform list of simple primitive types.


 RsslQos Reference Group
 

The quality-of-service primitive type can be used to indicate the quality of associated content using timliness and rate characteristics.


 RsslState Reference Group
 

The state primitive type can be used to convey data and stream health information.


 RMTES Decoder Reference Group
 

The RMTES(Reuters Multilingual Text Encoding Standard) Decoder can be used to decode RMTES content into UTF8 or UCS2 Unicode strings for display or machine parsing purposes.
