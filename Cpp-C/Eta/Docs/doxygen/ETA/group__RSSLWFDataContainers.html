Transport API Data Container Structures
Transport API Wire Format Data Package

The Transport API Data Package's container structures are required to encode or decode OMM binary data. These structures are used to encode primitive and container Transport API data structures.

Modules

 RsslFieldList Reference Group
 

The Field List container type allows the user to represent a collection of field identifier - value pairs, where the value can be a container type or a primitive type. The field identifier can be cross referenced with a field dictionary to determine the specific type of the content.


 RsslElementList Reference Group
 

The Element List container type allows the user to represent a collection of name - type - value triples, where the value can be a container type or a primitive type. The name and type information is similar to the content that would be cross referenced with a field dictionary when using an RsslFieldList. No dictionary is required to decode RsslElementEntry contents.


 RsslMap Reference Group
 

The Map container type allows the user to represent a primitiveType key indexed collection of uniform entries (e.g. each entry houses the same containerType).


 RsslSeries Reference Group
 

The Series container type allows the user to represent an implicitly indexed collection of uniform entries (e.g. each entry houses the same containerType). This is well suited for sending infrequently or non-updating table like content, where each RsslSeriesEntry represents an additional row in a table.


 RsslVector Reference Group
 

The Vector container type allows the user to represent a RsslUInt indexed collection of uniform entries (e.g. each entry houses the same containerType). This container type can optionally support sorting, where the consumer of content is responsible for applying actions in the appropriate manner to maintain ordering.


 RsslFilterList Reference Group
 

The Filter List container type allows the user to represent an 8-bit ID indexed collection of non-uniform entries (e.g. each entry can house different container types).
