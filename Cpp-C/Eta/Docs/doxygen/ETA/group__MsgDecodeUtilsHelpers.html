RSSL_API RsslRet rsslAddGroupId ( RsslBuffer *  pGroupId,
RsslUInt16  groupIdToAdd 
)
Warning:
  • This should not be run on a group ID buffer contained in a decoded message, as this can cause data corruption.
  • The user must have allocated enough free space to the buffer to add another two byte identifier.
Parameters:
[in]groupIdToAddtwo byte ID to append to the groupID contained in pGroupId buffer
[out]pGroupIdRsslBuffer with any existing groupID information.
Returns:
On success, returns RSSL_RET_SUCCESS.
