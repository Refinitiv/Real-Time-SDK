Enumeration Type Documentation

enum RsslStatusFlags
See also:
RsslStatusMsg, RsslMsg
Enumerator:
RSSL_STMF_NONE 

(0x000) No RsslStateMsg flags are present

RSSL_STMF_HAS_EXTENDED_HEADER 

(0x001) Indicates that this RsslStatusMsg has an extended header buffer, contained in RsslStatusMsg::extendedHeader.

RSSL_STMF_HAS_PERM_DATA 

(0x002) Indicates that this RsslStatusMsg has permission data, contained in RsslStatusMsg::permData.

RSSL_STMF_HAS_MSG_KEY 

(0x008) Indicates that this RsslStatusMsg has a msgKey, contained in RsslStatusMsg::msgBase::msgKey.

RSSL_STMF_HAS_GROUP_ID 

(0x010) Indicates that this RsslStatusMsg has a GroupId, contained in RsslStatusMsg::GroupId

RSSL_STMF_HAS_STATE 

(0x020) Indicates that this RsslStatusMsg has stream or group state information, contained in RsslStatusMsg::state.

RSSL_STMF_CLEAR_CACHE 

(0x040) Indicates that all cached header or payload data associated with this item stream should be cleared.

RSSL_STMF_PRIVATE_STREAM 

(0x080) Indicates that this RsslStatusMsg is an acknowledgement of a private stream establishment. Or, if RsslStatusMsg::state::streamState's value is RSSL_STREAM_REDIRECTED, the presence of this flag indicates that the current stream can only be opened as a private stream.

RSSL_STMF_HAS_POST_USER_INFO 

(0x100) Indicates that this RsslStatusMsg has post user info, contained in RsslStatusMsg::postUserInfo.

RSSL_STMF_HAS_REQ_MSG_KEY 

(0x200) Indicates that this RsslStatusMsg has the original request's message key, contained in RsslStatusMsg::msgBase::reqMsgKey

RSSL_STMF_QUALIFIED_STREAM 

(0x400) Indicates that this RsslStatusMsg is an acknowledgement of a qualified stream establishment.
