RsslUInt Encoding
RsslUInt Reference Group

Detailed information about RsslUInt encoding. More...

Detailed Description

When encoding an RsslUInt, only the number of bytes necessary to represent the contained value will be encoded.

RsslUInt Encoding Example

The following example demonstrates encoding an RsslUInt.

Creation and initialization of the RsslEncodeIterator is not shown in the example, however see the example in Transport API Encode Iterator for more information. The example assumes that the encIter was properly cleared and has an appropriate size buffer and version information already set.

    //Create and populate the RsslUInt
    RsslUInt rUInt = 65535;
    RsslRet retVal;

    //Invoke rsslEncodeUInt() function to encode content
    if ((retVal = rsslEncodeUInt(&encIter, &rUInt)) < RSSL_RET_SUCCESS)
    {
        printf("Error %s (%d) encountered with rsslEncodeUInt.  Error Text: %s\n", 
                    rsslRetCodeToString(retVal), retVal, rsslRetCodeInfo(retVal));
    }
