RsslDate Reference Group
Transport API Data Primitive Types

The RsslDate Primitive Type is an optimized wire representation of date that can contain month, day, and year. More...

Detailed Description

The RsslDate functionality is defined in "rtr/rsslDateTime.h", "rtr/rsslPrimitiveEncoders.h", and "rtr/rsslPrimitiveDecoders.h"

RsslDate Primitive Type Overview

The RsslDate Primitive Type is an optimized wire representation of a date containing month, day, and year. This type allows for representing the entire date or individual portions as blank, for example having a valid month and year but a blank day. See RsslDate Primitive Type for more detailed information.


RsslDate Encoding Functions

The following function is used while encoding an RsslDate.
See RsslDate Encoding for more detailed information.

  • rsslEncodeDate() - Encodes an RsslDate primitive type. This will perform any necessary byte swapping when encoding into the buffer referred to by the RsslEncodeIterator.



RsslDate Decoding Functions

The following function is used while decoding an RsslDate.
See RsslDate Decoding for more detailed information.

  • rsslDecodeDate() - Decodes an RsslDate primitive type. This will perform any necessary byte swapping when decoding from the buffer referred to by the RsslDecodeIterator.



RsslDate Utility & Conversion Functions

Transport API C Edition provides several common conversion helpers for the RsslDate. These can help with comparing and converting between RsslDate and various date strings.
See RsslDate Utility & Conversion Functions for more detailed information.
