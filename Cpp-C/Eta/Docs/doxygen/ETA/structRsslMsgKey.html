RsslMsgKey Struct Reference
Transport API Message Key

The RsslMsgKey contains a variety of attributes used to identify the contents flowing within a particular stream. This information, in conjunction with domainType and quality of service information, can be used to uniquely identify a data stream. More...

#include <rsslMsgKey.h>

Data Fields

 Container Type of the msgKey attributes. Must be a container type from the RsslDataTypes enumeration.
RsslBuffer encAttrib
 Encoded MsgKey attribute information, used for additional item stream identification attributes. Contents are typically specified in the domain model. Type is specified by attribContainerType.
RsslUInt32 filter
 Filter specification, used to request or indicate which filter entries are present with a filter list payload.
RsslUInt16 flags
 Flag values used to indicate optional member presence. The available options are defined by values present in RsslMsgKeyFlags.

 User specified numeric identifier. Usage of this is defined on a per-domain model basis.
RsslBuffer name

 Name Type. This indicates the type of the name member, examples are ‘User Name’ or ‘RIC’ (Reuters Instrument Code).
RsslUInt16 serviceId
 The serviceId is a two-byte unsigned integer used to identify a specific service.
