System Types
Transport API Data Primitive Types

The system types are platform independent representations of system types (e.g. 8 bit signed integer, 16 bit unsigned integer, etc.). More...

Detailed Description

These definitions leverage some common platform type defines contained in "rtr/os.h" which is a header file used to streamline cross-platform development. The system types are provided for use with Transport API interfaces and internal implementation.

Note:
Some system types may not have a specific corresponding RWF type for encoding or decoding.
