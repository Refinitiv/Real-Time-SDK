RSSL Server Operations
Transport API

The RSSL Server structure is used to create and maintain a listening socket that allows the application to accept connections created with rsslConnect.

Modules

 RSSL Bind Operation
 

The rsslBind function establishes a listening socket connection that can be used for both standard and HTTP rsslConnect connections.

RSSL_API RsslInt32 rsslServerBufferUsage ( RsslServer *  srvr,
RsslError *  error 
)

Typical use:
This function can be called to find out the number of used buffers for the server. This, in combination with the bufferPoolSize used as input to the rsslBind call, can be used to monitor and potentially throttle buffer usage.

Parameters:
srvrRSSL Server to obtain buffer usage for
errorRSSL Error, to be populated in event of an error
Returns:
RsslInt32 If less than 0, this is an RsslRet value, otherwise it is the total number of buffers in use by the server
