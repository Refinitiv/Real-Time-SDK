Generic XML Schema for Non-interactive Provider Group
The generic XML schema for NiProviderGroup is as follows:
 
 
<NiProviderGroup>
    <DefaultNiProvider value="VALUE"/>
    <NiProviderList>
        <NiProvider>
            <Name value="VALUE"/>
            ...
        </NiProvider>
    </NiProviderList>
</NiProviderGroup>
