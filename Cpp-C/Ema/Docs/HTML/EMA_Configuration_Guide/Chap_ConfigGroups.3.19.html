Dictionary Entry Parameters
Use the following parameters when configuring a Dictionary entry in the EMA.
 
Table 11: Dictionary Group Parameters  
Parameter Name
Type
Default
Notes
DictionaryType
Enumeration
ChannelDictionary
Specifies the dictionary loading mode.
Use enumeration values with EMA’s programmatic configuration (for details, refer to in Section 4.4)
Possible values are:
•
FileDictionary (0): The EMA loads the dictionaries from the files specified in the parameters RdmFieldDictionaryFileName and EnumTypeDefFileName.
•
ChannelDictionary (1): The EMA downloads dictionaries by requesting the dictionaries from the upstream provider.
EnumTypeDefFileName
EmaString
 
Sets the location of the EnumTypeDef file.
Name
EmaString
 
Sets a unique name for a Dictionary component in the DictionaryList.
RdmFieldDictionaryFileName
EmaString
 
Sets the location of the RdmFieldDictionary.
