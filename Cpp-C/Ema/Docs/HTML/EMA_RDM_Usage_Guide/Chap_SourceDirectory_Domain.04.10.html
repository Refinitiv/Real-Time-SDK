Source Directory Group Filter Entry
The Group filter entry (SERVICE_GROUP_FILTER, SERVICE_GROUP_ID) conveys item group status and item group merge information. Every item stream is associated with an item group as defined by the ItemGroup provided with the item’s RefreshMsg or StatusMsg. If some kind of change impacts all items within the same group, only a single group status message need be provided. For more information on item group use and handling, see the Message API C++ Edition Developers Guide.
If multiple group FilterEntrys are received in a single FilterList, then they should be applied in the order in which they were received.
Any default behavior is explained in the Element’s description.
