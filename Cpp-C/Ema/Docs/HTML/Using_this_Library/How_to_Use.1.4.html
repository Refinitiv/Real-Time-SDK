Toolbar Buttons
Online Documentation includes the following toolbar buttons which help you navigate and interact with the product’s documentation:
: This is the Show in Contents button, designed for use with the search feature. When in the Search tab in the left-side navigation, clicking this button brings up the Contents tab (in the left-side navigation) and highlights the current topic. If you are already in the Contents tab, behavior is browser dependent:
•
In IE, clicking this button highlights the current topic (highlighted).
•
Mozilla supports this function only when using the search.
: This is the Show Previous Topic button. Clicking this button loads the preceding topic (according to the Contents tab). If you are at the first topic in a particular book, and click this button, you will load the last topic of the manual that precedes this one in the Contents tab. For further details on navigating topics, refer to Navigating Topics in the Contents Tab.
: This is the Show Next Topic button. Clicking this button loads the next topic (according to the Contents tab). If you are at the last topic in a particular book, and click this button, you will load the first topic of the next manual in the Contents tab. For further details on navigating topics, refer to Navigating Topics in the Contents Tab.
: This is the Print Topic button. Clicking this button will print the currently displayed topic.
