How to build & run

1. Copy this file to some EMAC example source folder (e.g copy to IProv100)
2. Rename original example source file (e.g. IProvider.cpp -> IProvider.cpp_old)
3. Rename ContainerPerformance.cpp to the example name (e.g. ContainerPerformance.cpp -> IProvider.cpp)
4. Build current example (e.g build IProv100)
5. Run built example (e.g run IProv100)
