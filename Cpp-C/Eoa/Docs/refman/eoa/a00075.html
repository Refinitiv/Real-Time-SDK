Detailed Description

OmmDateTime encapsulates year, month, day, hour, minute, second, millisecond, microsecond and nanosecond information.

OmmDateTime is a read only class. All methods in this class are Lock-free.

See also:
Data, EoaString, EoaBuffer, OmmMemoryExhaustionException
