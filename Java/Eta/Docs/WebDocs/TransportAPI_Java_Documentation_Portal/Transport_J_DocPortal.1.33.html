Value Added Component Examples
 
Example
Description
Consumer
Consumer is an example of an OMM consumer written with the ValueAdd library and demonstrates how to consume data from an OMM Provider using Value Added Components.
NIProvider
NIProvider is an example of an OMM non-interactive provider written with the ValueAdd Library and demonstrates how to non-interactively provide Level I Market Price data to an Advanced Data Hub (ADH) using Value Added Components.
Provider
Provider is an example of an OMM interactive provider written with the ValueAdd Library and demonstrates how to interactively provide data to OMM consumers using Value Added components.
WatchlistConsumer
WatchlistConsumer is an example of an OMM consumer written with the ValueAdd package and demonstrates how to consume data from an OMM provider that uses the built-in watchlist feature of the Value Added Components. When enabled, various behaviors of recovery and aggregation are performed that you would otherwise need to implement yourself.
