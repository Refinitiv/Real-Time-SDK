Codec
CodecExample is a high-level, basic demonstration of encoding and decoding. You can set break points and step through the code to get a look and feel for how to use the Transport API in different ways.
NewsViewer
NewsViewer is a display application that you can use to view news headlines and open stories in various languages and character sets. NewsViewer is also a high-level application that demonstrates decoding and correctly interpreting content using the Reuters Multilingual Text Encoding Standard (RMTES).
