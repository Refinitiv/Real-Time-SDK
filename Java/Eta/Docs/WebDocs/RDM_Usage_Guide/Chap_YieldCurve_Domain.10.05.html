Yield Curve Update Message
A Yield Curve Update Message is sent by OMM provider and OMM non-interactive provider applications. It conveys any changes to an item’s data.
 
Warning! Although the payload is an FieldList, some field entries are sent as more complex types such as Vector and Array. Encoding and decoding applications should be aware of this and ensure proper handling of these types.

Optional. If a provider is sending a conflated update, conflationTime specifies the time interval (in milliseconds) over which data is conflated.
