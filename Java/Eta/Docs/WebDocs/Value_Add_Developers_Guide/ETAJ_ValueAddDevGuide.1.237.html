Dictionary Request Members
MEMBER
DESCRIPTION
rdmMsgType
Required. Dictionary message type. For a dictionary request, send DictionaryMsgType.REQUEST.
flags
Required. Indicates the presence of optional dicitonary request members. See Table 147 for details.
serviceId
Required. Specifies the service from which to request the dictionary.
verbosity
Required. Indicates the amount of information desired from the dictionary. Available values are:
•
Dictionary.VerbosityValues.INFO = 0x00: Version information only.
•
Dictionary.VerbosityValues.MINIMAL == 0x03: Provides information needed for caching.
•
Dictionary.VerbosityValues.NORMAL == 0x07: Provides all information needed for decoding.
•
Dictionary.VerbosityValues.VERBOSE == 0x0F: Provides all information(including comments).
Providers are not required to support the MINIMAL and VERBOSE filters.
dictionaryName
Required. Indicates the name of the requested dictionary.
