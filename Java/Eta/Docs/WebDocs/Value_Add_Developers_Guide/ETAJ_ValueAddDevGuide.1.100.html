ReactorChannel.packBuffer Return Values
The following table defines return and error code values that can occur when using ReactorChannel.packBuffer.
RETURN CODE
DESCRIPTION
Positive value or ReactorReturnCodes.SUCCESS
Success Case
The amount of available bytes remaining in the buffer for packing.
Negative value
Failure Case
This value indicates that some type of failure has occurred. If the FAILURE return code is returned, the ReactorChannel should be closed.
This method calls the Channel.packBuffer method which has its return values described in the Transport API Java Developers Guide.
