TunnelStreamAcceptOptions
CLASS MEMBER
DESCRIPTION
statusEventCallback
Specifies the instance of the callback for TunnelStreamStatusEvents, which provides the TunnelStream on initial connection and then communicates state information about the tunnel stream afterwards.
For details on TunnelStreamStatusEventCallback, refer to Section 6.6.4.1.
defaultMsgCallback
Specifies the instance of the callback that will handle all other content received on this TunnelStream.
For details on TunnelStreamDefaultMsgCallback, refer to Section 6.6.4.1.
userSpecObject
Specifies a user-defined object passed in via these options and then associated with the
TunnelStream.
classOfService
A ClassOfService with members indicating behaviors the application wants to assign to the TunnelStream.
For more information on class of service, refer to Section 6.6.3.
guaranteedOutputBuffers
Sets the number of pooled buffers available to the application when writing content to the TunnelStream.
