Reactor Destruction
When the application no longer requires a Reactor instance, it can destroy it using the following method.
 
Table 9: Reactor Destruction Method  
Method NAME
DESCRIPTION
shutdown
Shuts down and cleans up a Reactor. This also sends ReactorChannelEvents, indicating channel down, to all ReactorChannels associated with this Reactor.
