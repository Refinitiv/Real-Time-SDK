Setting Bookmarks (Firefox Browser Only)
When using the Firefox browser, you have access to an additional tab in the left-side pane called Favorites.
If you access a topic frequently and want to make it easier to find, you can add it to the Favorites tab. Simply access the topic, and then in the Favorites tab, click Add. The browser will add the topic as a link in the Favorites tab. To access this topic in the future, simply click this link.
