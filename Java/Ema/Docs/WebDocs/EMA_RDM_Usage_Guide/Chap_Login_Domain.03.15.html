Login Ack Message
OMM provider applications encode and send acknowledgment messages (AckMsg) to acknowledge the receipt of Post messages. This message is used whenever a consumer is posting off-stream and asks for acknowledgments. The acknowledgment contains a positive (ACK) or negative (NACK) code. For more details on posting, see the Message API Java Edition Developers Guide.
